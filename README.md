# Flight Search App UI - Flutter
A beautiful, modern, and clean **Flight Search UI** built using Flutter. This project showcases a responsive form design for booking flights, complete with interactive custom widgets such as date/time pickers, trip type selection, and customized dropdowns.
## 🚀 Features
 * **Trip Type Selection:** Clean tab/toggle buttons to switch between **Round Trip**, **One Way**, and **Multi-City** modes.
 * **Dynamic Location Input:** Elegant input fields for selecting origin ("From") and destination ("To") with custom icons.
 * **Custom Pickers:**
   * Interactive **Time Picker** wheel.
   * Native Material **Date Picker** dialog integration.
 * **Styled Dropdowns:** Formatted dropdown selection menus for **Flight Class** (First, Business, Economy) and **Passenger Count**.
 * **Modern Material Design:** Uses a dark-themed aesthetic with prominent CTA (Call to Action) "Search Flights" button.
 * **Debug Banner Ready:** Clean layout constraints handling various screen states seamlessly.

## 🏃‍♂️ Getting Started
To get a local copy up and running, follow these simple steps:
### Prerequisites
Make sure you have the Flutter SDK installed on your machine.
### Installation
 1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/flight-search-ui-flutter.git
   
   ```
 2. Navigate into the project directory:
   ```bash
   cd flight-search-ui-flutter
   
   ```
 3. Get the required dependencies:
   ```bash
   flutter pub get
   
   ```
 4. Run the app:
   ```bash
   flutter run
   
   ```
